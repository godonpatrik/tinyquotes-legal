# tinyquotes-legal

Static legal pages for the **Tiny Quotes** iOS app, served via GitHub Pages.

- Privacy Policy — `/privacy/`
- Terms of Use — `/terms/`

Live at: https://godonpatrik.github.io/tinyquotes-legal/

## Editing

Plain HTML, no build step. Edit the `index.html` files and push to `main`;
GitHub Pages redeploys automatically.

The in-app links live in the Tiny Quotes repo at `lib/legal.ts`.
